# Supabase-php
A site in PHP that integrates with Supabase to take advantage of their auth server and data storage

# Instalation
### Prerequisites
- [Composer](https://getcomposer.org/download/)
- [XAMPP](https://www.apachefriends.org/)
- [Node](https://nodejs.org/en/)
### Setup
```
composer install
npm install
```
